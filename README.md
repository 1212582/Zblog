Zblog
=======

a java blog
---------------

### 核心功能
1.	文章/页面发布、分类、统计阅读次数。
2.	提供文章形式的RSS聚合。
3.	提供链接的添加、归类功能。
4.	附件上传添加管理功能。
4.	评论的管理，垃圾信息过滤功能。
5.	文章静态html页面生成。
6.	会员注册登录，后台管理功能。
7.	lucene实现的站内搜索。
8.	防xss、csrf攻击。

### 技术选型

#### 后端
* Ioc容器 Spring
* Web框架 SpringMVC
* Orm框架 MyBatis
* 安全权限 Shiro
* 搜索工具 Lucene
* 缓存 Ehcache
* 静态化 FreeMarker
* 视图模板 Jsp/Jstl
* 其它 Jsoup(xss过滤)、fastjson、IKAnalyzer

#### 前端
* jquery js框架
* bootstrap 后台界面
* ueditor 编辑器
* webuploader 文件上传
* font-wesome 字体/图标

### 支持浏览器
* 前台 ie6+
* 后台 chrome/firefox/ie8+
