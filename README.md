### 核心功能

### 技术选型

#### 后端
* Ioc容器 Spring
* Web框架 SpringMVC
* Orm框架 MyBatis
* 搜索框架 Lucene
* 缓存 Ehcache
* 静态化 FreeMarker
* 其它 Jsoup(xss过滤)、fastjson

#### 前段
* jquery js框架
* bootstrap 界面
* ueditor 编辑器
* webuploader 文件上传
* font-wesome 字体/图标

### 支持浏览器
* 前台 ie6+
* 后台 chrome/firefox/ie8+
